试用hexo

0.node&npm

node.js 官网地址 https://nodejs.org/en/

1. 安装

https://hexo.io/zh-cn/docs/
npm install -g hexo-cli

如果出错：$ npm install hexo –no-optional
[fsevents] Success: “/usr/local/lib/node_modules/hexo-cli/node_modules/hexo-fs/node_modules/chokidar/node_modules/fsevents/lib/binding/Release/node-v46-darwin-x64/fse.node” is installed via remote

/usr/local/bin/hexo -> /usr/local/lib/node_modules/hexo-cli/bin/hexo

2. 初始化本地站点（folder在~/github/hexo）

﻿$ hexo init

$ cd

$ npm install

3. 启动本地服务

hexo server

之后本地4000端口号即可看到页面。

4. 生成静态页面

hexo new “试用hexo”
hexo generate
